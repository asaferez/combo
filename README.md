# COMBO — מעצב ארונות

כלי דפדפן פשוט ומהיר לאנשי המכירות של [קומבו](https://com-bo.co.il), לבניית ארונות מול הלקוח בזמן אמת.

## הפעלה

פותחים את הקובץ [`combo.html`](combo.html) בכל דפדפן — אין צורך בהתקנה או בשרת. הכל קובץ אחד עצמאי.

## יכולות

- הוספת עמודות לארון והגדרת רוחב וגובה לכל עמודה (בס"מ)
- חלוקת כל עמודה למדורים: **פתוח**, **מדף** (עם מספר מדפים), **תלייה**, **מגירה** (עם מספר מגירות וחזית זכוכית)
- תצוגה דו-ממדית חיה (חזית הארון) שמתעדכנת מיידית עם כל שינוי
- מעקב אחר הגובה שנותר בכל עמודה
- שמירה אוטומטית בדפדפן (localStorage)
- **ייצוא** ל-JSON ו**ייבוא** של פרויקט שמור — לצורך המשך עבודה בכלי תלת-הממד המקצועי

## פורמט הייצוא

הקובץ נשמר כ-JSON עם רשימת העמודות, רוחב, גובה כולל, והמדורים שבכל עמודה — מבנה נוח להזנה לתוכנות עיצוב/רינדור.
